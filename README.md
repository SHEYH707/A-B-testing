# A/B-тестирование для показателей кликабельности (CTR)

Локальное приложение на Python для анализа A/B-экспериментов по показателю CTR. Программа загружает CSV-файл, проверяет и очищает данные, рассчитывает CTR для вариантов A и B, выполняет статистическую проверку различий, строит графики и сохраняет итоговый отчёт.

## Установка

Откройте Anaconda Prompt и перейдите в папку проекта:

```bash
cd путь_к_папке_A_B_tests
```

Создайте окружение:

```bash
conda env create -f environment.yml
```

Активируйте окружение:

```bash
conda activate A_B_tests
```

## Запуск

```bash
python main.py
```

После запуска откроется графическое окно приложения.

## Формат входного CSV-файла

Приложение работает с пользовательским форматом данных, где одна строка соответствует одному показу пользователю.

Обязательные столбцы:

| Столбец | Описание |
|---|---|
| `user_id` | идентификатор пользователя |
| `timestamp` | дата и время показа |
| `group` | группа эксперимента: `control` или `treatment` |
| `landing_page` | показанная страница: `old_page` или `new_page` |
| `converted` | факт клика по целевому элементу: `0` или `1` |

В рамках проекта `converted = 1` интерпретируется как клик пользователя по целевому элементу интерфейса. Поэтому количество кликов рассчитывается как сумма значений `converted`, а CTR рассчитывается по формуле:

```text
CTR = clicks / impressions * 100%
```

Вариант A соответствует группе `control`, вариант B соответствует группе `treatment`.

## Порядок работы

1. Нажать кнопку **Загрузить датасет** или выбрать пункт меню **Файл -> Загрузить CSV**.
2. Выбрать CSV-файл, например `data/ab_data.csv`.
3. Нажать **Выполнить анализ**.
4. Посмотреть статистический отчёт в отдельном окне.
5. Нажать **Построить графики** для просмотра визуализаций.
6. Нажать **Сформировать отчёт** для сохранения итоговых файлов.

## Итоговые файлы

После формирования отчёта в папке `output` создаётся каталог вида:

```text
report_YYYYMMDD_HHMMSS
```

Внутри сохраняются:

- `analysis_report.txt` — текстовый отчёт
- `cleaned_data.csv` — очищенный датасет
- `summary.csv` — общая статистика по вариантам
- `daily_metrics.csv` — дневные показатели CTR
- `ctr_comparison.png` — столбчатая диаграмма сравнения CTR
- `ctr_distribution.png` — гистограмма распределения дневного CTR
- `ctr_by_date.png` — диаграмма рассеяния CTR по датам
- `ctr_boxplot.png` — boxplot дневного CTR

## Тестирование

```bash
python -m pytest
```

Тесты находятся в папке `tests`.
